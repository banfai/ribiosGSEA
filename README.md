*ribiosGSEA*: Tools for gene-set analysis
===

## What is *ribiosGSEA*?

*ribiosGSEA* is a R package that assists gene-set analysis of various kinds.


## Installation

Run following commands in the R console.

```{R}
library(devtools)
devtools::install_github("bedapub/ribiosGSEA")
```
